# HomeLab 2: CICD Practise by Deploying a PHP Gallery Webpage on a Local Laptop's Resources.
## Introduction
<p align="justify">
In this project, I created a a simple webpage that users can upload photos with description as a journal-like record keeping tool. The main purpose of this project is to learn the CICD's practise by using Git as the versioning tool for the CI's part and docker as the resource deployment tool for the CD's part. All the resources are from local laptop.
</p>

## Project Objective
<p align="justify">
- To practice simple CICD versioning and deployment's practice by using Git and Docker.
</p>
